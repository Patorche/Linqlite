Linqlite n’a pas pour ambition de gérer l’administration d’un pays.  
Ce n’est pas un framework tentaculaire, ni une usine à gaz destinée à des bases distribuées ou à des architectures gigantesques.

Linqlite est une **librairie** — volontairement simple, légère et maîtrisable — pensée pour les petites bases locales, les applications desktop, les outils personnels ou professionnels qui ont besoin d’un ORM fiable sans la lourdeur d’un framework complet.

Il ne cherche pas à tout faire.  
Il cherche à **bien faire ce qui compte** : mapper des objets, générer du SQL propre, et faciliter le quotidien du développeur.
